# dotfiles-alt

Ficheiros específicos da sessão **HyprTheme** (hypr-alt).

| Path | Destino na instalação |
|------|------------------------|
| `session/run-hyprland-theme` | `/usr/bin/run-hyprland-theme` |
| `session/hyprland-theme.desktop` | `/usr/share/wayland-sessions/` |

Configs Hyprland ficam em `~/.config/hyprtheme/` (geradas pelo install a partir de `shared-themes/hypr/`).
