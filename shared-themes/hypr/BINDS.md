# Hyprland — Cheatsheet de atalhos

Abrir: **Super+F1** · Ficheiro: `~/.config/themes/hypr/BINDS.md`

---

## Geral

| Tecla | Ação |
|-------|------|
| Super+Return | Terminal (foot) |
| Super+Shift+Return | Terminal flutuante (`foot-float`) |
| Super+Q | Fechar janela |
| Super+M | Sair do Hyprland |
| Super+Esc | Sair do Hyprland (exit) |
| Super+E | Thunar (ficheiros) |
| Super+D | Launcher de apps (rofi) |
| Super+Shift+P | Pseudotile |
| Super+F1 | **Este cheatsheet** |

## Janelas

| Tecla | Ação |
|-------|------|
| Super+← ↑ ↓ → | Focar janela |
| Super+F | Maximize toggle (`fullscreen, 1`) |
| Super+F5 | Float + centrar |
| Super+F7 | Toggle split |
| Super+F8 | Float + pin + centrar |
| Alt+Tab | Trocar com próxima janela |
| Alt+Shift+Tab | Trocar com janela anterior |

## Theme engine

| Tecla | Ação |
|-------|------|
| Super+Shift+R | Recarregar config Hyprland |
| Super+T | Seletor de tema |
| Super+F10 | Wallpaper (sem mudar cores) |
| Super+Shift+T | Wallpaper + Matugen |
| Super+Shift+C | Menu de configuração |
| Super+X | Power menu (rofi) |

## Apps e utilitários

| Tecla | Ação |
|-------|------|
| Alt+V | Histórico do clipboard |
| Super+Shift+D | Window switcher (rofi) |
| Super+P | Menu de screenshot |
| Super+Print | Screenshot da área (slurp) |
| Print | Screenshot da janela ativa |
| Alt+Print | Screenshot do desktop |
| Super+W | Firefox |
| F12 | Reiniciar waybar |
| Super+F12 | Reiniciar waybar (alt) |

## Workspaces

| Tecla | Ação |
|-------|------|
| Super+1 … 0 | Ir para workspace |
| Super+Shift+1 … 0 | Mover janela para workspace |
| Super+S | Scratchpad `magic` |
| Super+Shift+S | Mover janela → scratchpad |
| Super+scroll ↓/↑ | Workspace anterior / seguinte |
| Gesto 3 dedos ↔ | Trocar workspace |

## Rato (Super + …)

| Tecla | Ação |
|-------|------|
| Botão esquerdo + arrastar | Mover janela |
| Botão direito + arrastar | Redimensionar |
| Scroll ↓/↑ | Workspace e+1 / e-1 |

## Multimédia

| Tecla | Ação |
|-------|------|
| XF86AudioRaiseVolume | Volume + |
| XF86AudioLowerVolume | Volume − |
| XF86AudioMute | Mute |
| XF86AudioMicMute | Mute microfone |
| XF86MonBrightnessUp | Brilho + |
| XF86MonBrightnessDown | Brilho − |
| XF86AudioNext / Prev | Faixa seguinte / anterior |
| XF86AudioPlay / Pause | Play / pause |
