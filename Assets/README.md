# Assets vendored (offline install)

| Ficheiro | Conteúdo |
|----------|----------|
| `catppuccin-frappe-blue.zip` | GTK theme Catppuccin Frappe Blue |
| `catppuccin-frappe-blue-standard+default.zip` | alternativa (dot Assets) |
| `MacTahoe.tar.xz` | Ícones MacTahoe-dark |
| `Qogir-cursors.tar.xz` | Cursor Qogir |
| `catppuccin-frappe-blue-sddm.zip` | Tema SDDM |
| `Manhattan.zip` | GTK Manhattan (legado) |
| `.fonts/` | Fontes do dot (opcional) |

```bash
bash Assets/vendor-assets.sh          # temas + ícones + cursor + SDDM
bash Assets/vendor-assets.sh --fonts  # inclui dotfiles/fonts/.fonts
```

Também em `install/TO_CURSOR/` (gtkthemes + zip de referência).
